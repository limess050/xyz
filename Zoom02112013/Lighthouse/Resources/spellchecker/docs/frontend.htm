To call the front end on a textarea, consider the following example...

<form name"myFormName">
  <textarea name="myFieldName">Spelllde Wrang</textarea>
  <input onClick="doSpell()" type="button" value="Check Spelling">
</form>

The following script could be used to open the spell checking window...

<script language="javascript">
//called when the Check spelling button is clicked
function doSpell() {
    window.open("/spellchecker/window.cfm?jsvar=document.myFormName.myFieldName.value", null, 
        "height=230,width=450,status=no,toolbar=no,menubar=no,location=no"); 
}
</script>

The following things should be noted...
  • The example above assumes that your front end has been installed in a directory called spellchecker directly under the web server root (/spellchecker).
  • window.cfm?jsvar=xxx - is the page you should open up in a new window. jsvar should be a javascript variable that points to the content that should be spell checked.
