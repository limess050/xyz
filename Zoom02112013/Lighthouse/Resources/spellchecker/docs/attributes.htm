<CFX_JSpellCheck> Attribute Reference

Attribute

Type

State

Description

clx

text (path)

REQUIRED

Compressed Lexicon File, path to the file in the lex directory with the extension .clx usually c:\CFX_JSpellCheck\lex\american.clx for the american dictionary.

tlx

text (path)

REQUIRED

Text Lexicon File, path to the file in the lex directory with the extension .tlx usually c:\CFX_JSpellCheck\lex\american.tlx for the american dictionary.

userdict

text (path)

REQUIRED

User Dictionary File, path to the file which will contain a user defined dictionary. This is usually c:\CFX_JSpellCheck\lex\userdic.tlx.
words text   This is a string of words that will be spell checked.
action text check or add Used to distinguish between checking words, or adding a word

searchdepth

integer

1 to100

An integer specifing the search depth, this attribute has significant impact on the spell checkers response time. Default value is 50, high values conduct a more in depth search.

englishphonetic

boolean

default:false

Should be set to true if you are using the English language.  This option will use an English Phonetic Comparator for searches.  This comparator searches by phonetics, which often produces better results, and a few tests show that it has a higher return time.

format

text

javascript or wddx< /P>< /P>< /P>

Returns either a set of Java Script arrays of mispelled words, or output in WDDX XML Format, WDDX can easily be read by ColdFusion Application Server.  Default is JavaScript

multilingual

boolean

 < P> default: false< /P>

PLEASE NOTE: this is a server wide setting, the value used here, could effect other spell checkers on your server.  If you are only using one language PER SERVER then set this attribute to false.  It allows us to cache the SpellChecker object, providing a huge performance gain, by reusing objects.  When set to true, CFX_JSpellCheck will create a new object each time the tag is called, resulting in approximate performance loss of 30-60%. 

striphtml

boolean

default: true

Removes all tags from the words input attribute.  It will also remove anything inside <script>javascript</script> or <style>css</style> tags.

variable

text

    

The name of the variable the output of the tag will be stored in.  If this attribute is not specified, then it will output the result where <CFX_JSpellCheck> is called.

lexdir

text (path)

   

This attribute is a shortcut for specifying the tlx, clx, and userdict attributes.  Just put the path to the lex dir, and it will assume american.clx, american.tlx, and userdic.tlx for the clx, tlx, and userdic attributes respectivly.

word

text

 action= add

A word to add to the user dictionary when type=add is specified.
